## 🌐 如何让他人也能访问？

### 方案一：免费静态托管（推荐）

#### 1. GitHub Pages（最简单）
```bash
# 1. 创建GitHub仓库
# 2. 上传 index.html 文件
# 3. Settings → Pages → 选择 main branch → Save
# 4. 访问 https://你的用户名.github.io/仓库名
```

#### 2. Vercel（免费，无需Git）
1. 访问 https://vercel.com
2. 注册/登录
3. New Project → Upload → 拖入 index.html
4. 获得永久免费URL

#### 3. Netlify（拖拽上传）
1. 访问 https://netlify.com
2. Drag & Drop → 上传 index.html
3. 立即获得可分享链接

---

### 方案二：内网穿透（让局域网用户访问）

```bash
# 使用 ngrok（免费账号每月100小时）
ngrok http 8080
# 会生成一个公网URL，别人也能访问
```

---

### 方案三：部署到云服务器

如果学校/公司有服务器，可以：

1. 上传 `index.html` 到服务器的 `/var/www/html/` 目录
2. 配置 Nginx：

```nginx
server {
    listen 80;
    server_name your-domain.com;
    
    location / {
        root /var/www/html;
        index index.html;
    }
}
```

3. 重启 Nginx：`sudo systemctl restart nginx`

---

### ❌ 不推荐：微信小程序

微信小程序**不适合**这个项目，原因：
- WebGL 支持有限
- 需要原生开发适配
- 3D渲染性能差
- 开发成本高

**推荐**：使用H5/微信内置浏览器直接打开链接即可

## 🎨 如何自定义内容？

### 修改产品信息

编辑 [index.html](file:///d:/tstst/index.html) 中的 `assemblySteps` 数组：

```javascript
const assemblySteps = [
    {
        id: 1,
        title: '你的步骤名称',
        description: '详细说明文字',
        part: 'part_name'  // 对应3D零件
    },
    // ... 更多步骤
];
```

### 修改产品名称和型号

在HTML中找到这部分：

```html
<h3>你的产品名称</h3>
<p>型号: XXX-123</p>
<p>零件数: N件</p>
```

### 修改3D模型

在JavaScript的 `createMotorAssembly()` 函数中，可以：

#### 1. 修改颜色
```javascript
const metalMaterial = new THREE.MeshStandardMaterial({
    color: 0x3a4a5a,  // 修改RGB颜色值
    roughness: 0.4,
    metalness: 0.85
});
```

#### 2. 修改零件尺寸
```javascript
// CylinderGeometry(radiusTop, radiusBottom, height, segments)
const baseCylinder = new THREE.Mesh(
    new THREE.CylinderGeometry(1.0, 1.1, 0.25, 32),  // 调整参数
    metalMaterial
);
```

#### 3. 添加新零件
```javascript
// 创建新零件
const newPart = new THREE.Mesh(
    new THREE.BoxGeometry(1, 1, 1),
    metalMaterial
);
newPart.position.y = 1;
parts.new_part = newPart;
scene.add(newPart);
```

### 添加更多步骤

1. 在 `assemblySteps` 数组中添加新步骤
2. 在 `createMotorAssembly()` 中创建对应的3D零件
3. 记得在HTML中更新"零件数"
